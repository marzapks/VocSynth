VocSynth Help

Mic Mode

Tap Mic Mode and allow microphone access. VocSynth listens to your input level and pitch area, then triggers the selected synth patch. External microphones connected through a dongle or audio interface should work when the browser/device recognizes them as the current input.

Pitch Correction

Auto pitch correct rounds the detected vocal pitch to the nearest chromatic note for cleaner synth tracking.

Chord Builder

Tap Build Chord, then tap piano notes. The notes light up and appear as chips. Enable “Voice triggers built chord” to fire the whole chord from your voice.

Control Module

Use cutoff, resonance, ADSR, delay, reverb, LFO, EQ, drive, and stereo width to shape the sound.

Custom Uploads

Upload audio files to create custom sample textures. Browser security means full commercial VST/SF2 decoding is limited in a single HTML file, but the app accepts them as patch references.

Low Latency Note

For best response, use Safari/Chrome with no Bluetooth audio, close extra apps, and use wired headphones or an audio interface.
