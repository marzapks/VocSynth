VOCSYNTH GUIDE

DESIGNED BY MARIO GARCIA

QUICK START

Pick a sound bank tab (Lead / Bass / Strings / Wind), choose a patch from the 20-cell grid, then play notes on the piano keyboard. Use the octave buttons to shift the keyboard range.

MIC MODE

Tap MIC MODE to allow microphone access. Sing, hum, or whistle — VocSynth tracks your pitch in real time and re-voices it through the active patch. The HUD at the top shows the detected note and input level.

  • PITCH-CORRECT snaps incoming pitch to the nearest semitone (auto-tune style).
  • SMOOTH applies gentle glide between detected pitches; turn off for staccato tracking.
  • External mics work automatically — plug in your USB / Lightning / USB-C dongle mic before tapping MIC MODE so the browser sees it as the default input.

BUILD CHORD

Tap BUILD CHORD then tap notes on the keyboard — each note lights up and previews its sound. Tap an illuminated note again to remove it. The intervals are stored relative to the lowest note. With Mic Mode on, every voice-triggered note now plays the full chord transposed to your sung pitch. Use CLEAR ✕ to reset.

CUSTOM PATCHES

Tap + PATCH to upload audio files (wav, mp3, ogg, m4a, flac, aac, aif, aiff, webm, opus, wma). Each upload becomes a custom patch cell with a dashed border in any bank you upload it to. Selecting it routes both keyboard and mic triggers through your sample, pitch-shifted to follow the note.

CONTROL PANEL

  • Cutoff / Resonance — sweep the low-pass filter for classic synth movement.
  • Attack / Decay / Sustain / Release — shape every note's envelope.
  • Drive — analog-style saturation.
  • Master — overall output level.

FX · EQ · LFO · DELAY · REVERB

  • EQ — Low shelf, Mid bell, High shelf.
  • LFO — Set rate and depth, route to Pitch, Filter, or Amp.
  • Delay — Time, Feedback, Mix. Hit TAP TEMPO rhythmically (2+ taps) to set delay time by feel.
  • Reverb — Size and Mix for the space.

THEME

Tap the ◆ icon to cycle through neon palettes — every glow, frame and accent in the app re-tints instantly.

TIPS FOR BEST RESULTS

  • Sing clear, sustained vowels — pitch detection works best on stable tones.
  • Disable phone OS noise-suppression where possible for cleaner tracking.
  • For lowest latency: close other audio apps, use wired headphones, plug external mic in before launching.
  • VocSynth uses the Web Audio API — minimum end-to-end latency depends on your browser and device (typically 20–50 ms).
